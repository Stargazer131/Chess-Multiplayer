# Chess-Multiplayer

## Ứng dụng chơi cờ vua
+ Tự động ghép cặp 2 người chơi bất kỳ
+ Người chơi tự động thắng (và tự thoát) nếu người kia thoát
+ Khi kết thúc ván có thể bấm enter để chơi tiếp
+ Phải đợi cho đến khi được ghép cặp
